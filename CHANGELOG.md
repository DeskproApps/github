# Changelog

This project is following [Semantic Versioning](http://semver.org)

## [Unreleased][]

## [0.3.4][] - 2018-04-03

### Added

    - travis will atttach builds to Github PR's when enabled via s3 environment variables
    - adds @deskpro/redux-components

### Fixed

    - installer does not show the redirect url required to create an Oauth App in Github

### Changed

    - upgrade to @deskpro/apps-sdk-react version 0.2.13
    - upgrade to @deskpro/apps-dpat version 0.10.4
    - upgrade to @deskpro/apps-sdk-core version 1.0.0-beta.29  

## [0.3.3][] - 2018-03-26

- uses @deskpro/apps-sdk-react v0.2.12

## [0.3.2][] - 2018-03-23

- uses @deskpro/apps-sdk-react v0.2.11

## [0.3.1][] - 2018-03-16

- upgrades to @deskpro/redux-components v1.0.2

## [0.3.0][] - 2018-03-15

- uses @deskpro/redux-components

## [0.2.0][] - 2018-03-02

 - adds support for new custom fields

## [0.1.8][] - 2018-02-09

 - default `process.env.NODE_ENV` to `production` when packaging the app for distribution with webpack
